# Cross-Browser Testing Checklist

## Desktop (Latest 2 Versions)
- Chrome: layout, forms, live map, location prompts
- Firefox: typography rendering, focus states, form validation
- Safari: font loading, address search modal, map tiles
- Edge: dashboard tables, admin cards, buttons

## Mobile
- iOS Safari: location permission flow, sticky nav, form inputs
- Android Chrome: order flow, receipt upload, map tracking

## Notes
- Validate reduced-motion behavior (animations disabled).
- Confirm keyboard navigation and focus visibility on interactive elements.
- Verify contrast on cards, badges, and button states.
